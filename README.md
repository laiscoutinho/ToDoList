# 📌 Sistema Web de Gerenciamento de Tarefas

Projeto desenvolvido como exercício prático para treinar os conteúdos da disciplina **Programação Web**, do meu curso técnico em informática.  
O sistema permite o gerenciamento de **tarefas pessoais (CRUD)**.  

Tecnologias utilizadas:
- **Front-end:** React (JavaScript, HTML5, CSS3);
- **Back-end:** Spring Boot (Java);
- **Banco de Dados:** .
- **Arquitetura:** Cliente/Servidor (REST API)

---

## 🎯 Funcionalidades

### Tarefas
- Criar nova tarefa 
- Listar tarefas do usuário autenticado
- Atualizar tarefa
- Excluir tarefa

---

## 🛠️ Como Rodar o Projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/laiscoutinho/todolist
cd todolist
```

### 2. Rodar o Back-end (Spring Boot)

```bash
cd backend
./mvnw spring-boot:run
```
O servidor iniciará em:
👉 http://localhost:8080

### 3. Rodar o Front-end (React)

```bash
cd frontend
npm install
npm start
```
A aplicação abrirá em:
👉 http://localhost:3000
